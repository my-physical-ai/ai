태양광 핫스팟 드론 AI 검사

Teledyne FLIR + TOPA/AID 정부 실증 · DJI M30T/M4T
에너지·드론

드론이 항공 열화상으로 패널을 스캔하고, AI가 과열 패널을 자동 탐지·분류·위치 매핑합니다. 독일 정부 실증(FLIR A700+AI)에서 검사 비용을 패널당 €246 → €49(80% 절감)로 낮췄습니다. 결함 패널은 정상보다 10~20℃ 뜨겁습니다.

🖼️ 드론이 태양광 패널을 열화상 스캔 — 빨간 패널이 과열 결함

같은 문제에 5가지 모델이 모두 적용. YOLO(실시간 탐지), UNet 계열(경량 세그), Swin(미세결함), Autoencoder(라벨 없는 이상감지), 2단계 캐스케이드(패널 분리→결함 분류)까지.

FLIR 플라스틱 IR 용접 품질 검사

Teledyne FLIR A50/A70 · 자동차 대시보드·그릴
자동차 품질

자동차 플라스틱 부품은 적외선(IR) 용접으로 접합되는데, 균일 온도에 도달하지 않으면 접합 실패 → 리콜로 이어집니다. 육안으론 성공 여부를 알 수 없어, 라인에 고정 설치된 열화상이 온도 패턴을 실시간 감시해 상·하한을 벗어난 용접을 잡아냅니다.

🖼️ 고정 열화상 카메라가 컨베이어 위 용접부의 온도 균일도를 판정
🧠 사용 AI 모델 / 방식
Radiometric Thermal Streaming Threshold + Pattern ML GigE Vision / RTSP
FLIR A50/A70이 ±2℃ 정밀도로 라디오메트릭 데이터를 GigE Vision/RTSP로 스트리밍. 온도 패턴·그래디언트 정상 범위를 학습한 모델이 비균일·불완전 형상을 분류합니다.

로봇 용접 멜트풀 CNN-LSTM 감시

Therness HeatCore AI (2026) · 로봇 용접 셀 인라인
공정 품질

용접 품질을 "사후 부검"이 아닌 실시간 제어 루프로 전환. 고속 마이크로볼로미터 열화상 + 엣지 딥러닝이 기공·융합부족·과열을 100ms 이내에 잡습니다. 멜트풀(용융풀) 열 영상의 시계열 패턴으로 결함 형성 과정을 포착합니다.

🖼️ 멜트풀 열 영상이 CNN-LSTM을 거쳐 100ms 안에 결함 판정
🧠 사용 AI 모델
CNN-LSTM YOLO (30~120 FPS) Edge Deep Learning
CNN(공간 특징)+LSTM(시간 변화)으로 멜트풀 열 영상의 시계열 패턴 학습. 용융 금속 방사율 >0.7로 신호 지배적, 아크 간섭은 협대역 필터로 제거.
🔗 동영상 링크 & 검색어
  • 검색어:AI weld monitoring thermal weld pool computer vision
  • 검색어2:infrared thermography weld defect detection real-time
  • 출처:therness.com — AI weld monitoring
🎓
교육 포인트: CNN-LSTM은 빅맨님 BoT-SORT 추적 교육과 연결. 정적 탐지(CNN)에 시간 차원(LSTM)을 더하면 "결함 형성 과정"을 봅니다.

적층제조(3D프린팅) 결함 CNN 분류

학계 벤치마크 (ScienceDirect, 2026) · 21,127장 열화상
결함 검사

3D 프린팅은 한 층씩 쌓는데, 적층 중 결함(과/부족 압출, 층 밀림, 휨)이 내부에 숨습니다. 열화상이 매 층의 온도 분포를 실시간으로 보여줘 식기 전에 이상을 잡습니다. 7개 CNN을 동일 조건에서 벤치마크한 최신 연구입니다.

🖼️ 층별 열화상이 CNN을 거쳐 5가지 결함으로 분류 (정확도 97.26%)
🧠 사용 AI 모델 (7종 벤치마크)
Custom CNN (97.26%·최고) ResNet-18 (97.19%) MobileNetV2 (97.07%) EfficientNet-B0 VGG-16 / AlexNet
5개 결함(과/부족압출·층밀림·휨·정상)을 분류. Custom CNN이 정확도 1위 + 메모리 효율(1.91GB)로 엣지 배포 적합. MobileNetV2는 경량이라 CPU 실시간에 유리.
🔗 논문 링크 & 검색어
  • 검색어:deep learning defect detection additive manufacturing thermal imaging
  • 출처:sciencedirect.com — 3D프린팅 열화상 결함 벤치마크
🎓
"No Free Lunch"의 살아있는 예: 정확도 1위 Custom CNN과 경량 MobileNetV2 중 현장 하드웨어에 따라 선택이 갈립니다. NUC(CPU)라면 MobileNetV2 + OpenVINO가 정답.

모터·베어링 전이학습 고장진단

Deep Transfer Learning (ScienceDirect, 2023) · 3상 유도전동기
예지보전

베어링 고장·모터 권선 열화는 비계획 정지의 큰 비중. 열화상은 진동 분석이 이상을 확인하기 전에 마찰열·과열을 먼저 감지합니다. 비접촉·비침습·저비용이라 회전 기계에 이상적입니다.

🖼️ 전동기 베어링 핫스팟을 전이학습 CNN이 12가지 상태로 진단
🧠 사용 AI 모델
Deep Transfer Learning ADCNN (Attentive Dense CNN) ANN + FFT
사전학습 CNN(ImageNet)을 열화상으로 전이학습. 그레이스케일 강도 프로파일 + FFT + 신경망으로 12가지 상태(정상·베어링결함·회전자봉파손·정렬불량·기어마모)를 분류. 희소 데이터 극복.
🔗 논문 링크 & 검색어
  • 검색어:thermographic image diagnosis electrical motor deep transfer learning
  • 출처:sciencedirect.com — 모터 열화상 전이학습 진단
💡
왜 전이학습? 열화상 결함 데이터는 희소합니다. ImageNet 사전학습 가중치를 재활용하면 적은 데이터로도 높은 정확도를 달성 — 빅맨님 SAM3 라벨링→YOLO 증류와 같은 "데이터 절약" 발상.

🎯 핵심 정리 — AI 모델 관점

  • 같은 열화상, 다른 AI — 분류·탐지·이상·시계열에 따라 CNN·YOLO·AE·LSTM·전이학습이 갈린다.
  • 로봇 + 열화상이 대세 — POSCO Spot처럼 이동 플랫폼에 IR을 실어 사람이 못 가는 곳을 점검한다.
  • 이상감지(Autoencoder)가 예지보전의 핵심 — 정상만 학습하고 편차로 고장을 30~90일 전 예측.
  • 시간 차원이 결함 형성을 본다 — CNN-LSTM, 시계열 이상감지로 "변화 과정"을 포착.
  • 멀티모달이 안전을 완성 — 열화상 + OGI(가스) + 음향 + RGB 융합이 골든타임을 지킨다.
  • 빅맨님 파이프라인이 곧 정답 — YOLO→SAM3→AE→VLM이 이 사례들의 AI 모델 조합과 일치한다.
