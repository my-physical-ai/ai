1️⃣ 환경설정

1
USB 연결 확인

RealSense를 NUC의 USB 3.0 (파란색) 포트에 꽂고 인식되는지 확인합니다.

# RealSense 카메라가 USB에 인식되는지 확인
lsusb | grep -i intel

# 예상 출력:
# Bus 002 Device 003: ID 8086:0b3a Intel Corp. Intel(R) RealSense(TM) Depth Camera 435i
⚠️ USB 2.0에 꽂으면 안 됩니다. Depth 스트림이 정상 작동하지 않거나 FPS가 심하게 떨어집니다. 반드시 파란색(USB 3.0) 포트를 사용하세요.
2
conda 가상환경 활성화

기존 lerobot 가상환경을 재활용합니다. (새 환경 불필요)

# conda 초기화 (최초 1회 — 이미 했으면 스킵)
~/miniconda3/bin/conda init bash
source ~/.bashrc

# lerobot 가상환경 활성화
conda activate lerobot

# (lerobot) 프롬프트가 보이면 성공
💡 conda: 명령을 찾을 수 없습니다 오류 시 conda init bash 먼저 실행하세요.
3
USB 드라이버 + 패키지 설치

RealSense USB 커널 드라이버와 Python 패키지를 설치합니다.

# USB 커널 드라이버 설치 (apt — conda로는 불가)
sudo apt install -y libusb-1.0-0 libusb-1.0-0-dev

# Python 패키지 설치 (pip)
pip install pyrealsense2 flask opencv-python numpy

📝 주요 소스 설명

A
app.py — RealSense 초기화 핵심

Align은 반드시 설정해야 합니다. 없으면 클릭 좌표와 Depth 값이 어긋납니다.

# Depth를 Color 좌표계에 정렬 (필수!)
rs_align = rs.align(rs.stream.color)

# 매 프레임마다 정렬 적용
aligned = rs_align.process(frames)
color_frame = aligned.get_color_frame()
depth_frame = aligned.get_depth_frame()
B
app.py — 역투영 공식 (미션 3)

2D 픽셀 (u,v) + Depth Z → 3D 좌표 (X,Y,Z) 변환의 핵심 수학입니다.

# 역투영 공식: 2D 픽셀 → 3D 실제 좌표 (미터)
X = (u - cx) * Z / fx   # 좌우 위치 (미터)
Y = (v - cy) * Z / fy   # 상하 위치 (미터)
Z = depth                # 거리 (미터, 센서 값 그대로)

# cx, cy, fx, fy = 카메라 내부 파라미터 (자동 읽기)
C
app.py — 근접 경고 로직 (미션 7)

공장 안전존 적용의 핵심 패턴. ROI 내 유효 최소 거리로 위험 판정합니다.

# ROI 영역에서 Depth 데이터 추출
roi = depth_array[y1:y2, x1:x2].astype(np.float64) * rs_depth_scale

# 무효 데이터(0값) 제거 후 최소 거리 산출
valid = roi[roi > 0.1]   # 0.1m 이상만 유효
min_dist = float(np.min(valid))

# 임계 거리 이하면 경고
if 0 < min_dist < alert_dist:
    # 경고 시각 효과 + 알림 트리거
💡 alert_zone_pct(감지 영역 %)와 alert_dist_m(임계 거리)은 브라우저 슬라이더로 실시간 조절 가능합니다.
D
app.py — 설정값 수정 위치

app.py 상단에서 환경에 맞게 수정할 5개 값입니다.

🔧 문제 해결

❌ No RealSense device found
→ USB 3.0(파란색) 포트에 꽂았는지 확인
→ lsusb | grep Intel로 인식 확인
→ 다른 프로그램이 카메라 사용 중: sudo fuser /dev/video*
❌ Frame didn't arrive within 1000ms
→ test_realsense.py가 아직 실행 중이면 종료 후 재시도
→ USB 케이블을 뺐다 다시 꽂고 3초 대기
❌ Address already in use (포트 5000)
→ 이전 app.py가 아직 실행 중: sudo kill $(sudo lsof -ti :5000)
→ 또는 app.py 상단에서 FLASK_PORT = 5001로 변경
❌ 브라우저에서 영상이 안 나옴
→ 방화벽 확인: sudo ufw allow 5000
→ NUC IP 확인: hostname -I
→ 미션 카드를 클릭해야 RealSense가 초기화됩니다
❌ ModuleNotFoundError: pyrealsense2
→ conda activate lerobot 후 pip install pyrealsense2
❌ 클릭해도 거리가 0cm로 표시
→ Depth 센서 최소 거리 이내 (D435i: 약 10cm 이상 필요)
→ 반사가 심한 물체(거울, 금속 광택면)는 Depth 값이 0
