⚡ Step 1: YOLOE-26 모델 설치 (NUC)

YOLOE-26은 텍스트로 원하는 물체를 탐지하는 Open Vocabulary 모델입니다

1📦
ultralytics 업데이트YOLOE 지원 버전 필요
$pip install --upgrade ultralytics
▶ 이렇게 하면

YOLOE-26을 사용할 수 있는 최신 ultralytics가 설치돼요.

2⚡
YOLOE-26 다운로드 + OpenVINO 변환최초 1회만 실행
$cd ~/lerobot && python setup_yoloe.py
▶ 이렇게 하면

YOLOE-26s 모델을 다운로드하고 NUC CPU에서 빠르게 돌아가도록 OpenVINO로 변환해요.

1yoloe-26s-seg.pt 모델 파일이 자동 다운로드됩니다 (~100MB)
2set_classes()로 37종 탐지 대상(cup, bottle, box...)이 설정됩니다
3OpenVINO 포맷으로 변환되어 yoloe-26s-seg_openvino_model/ 폴더에 저장됩니다
📚
YOLO26 vs YOLOE-26 차이

YOLO26은 80종 사진을 외운 학생 — 외운 것만 인식. YOLOE-26은 영어 단어의 의미를 이해한 학생 — "cup"이라고 하면 어떤 컵이든 찾아요. 80종보다 적게 설정해도 더 정확합니다!

💡탐지 대상 변경: setup_yoloe.py의 DETECT_CLASSES 리스트를 수정하면 원하는 물체를 추가/제거할 수 있습니다. 수정 후 다시 실행하면 새 클래스가 적용돼요.

🎯 Step 2: SAM3 서버 시작 (4090)

SAM3는 물체의 정확한 픽셀 경계(윤곽선)를 그려주는 정밀 분할 모델입니다

3🔧
SAM3에 cutout 기능 추가배경 제거 이미지 생성 (최초 1회)
$cd ~/lekiwi_sam3/4090 && python patch_sam3_cutout.py
▶ 이렇게 하면

SAM3 서버에 배경을 흰색으로 제거하는 cutout 기능이 추가돼요.

1기존 sam3_server.py를 자동 백업합니다
2/segment 응답에 cutout_image 필드가 추가됩니다
4🎯
SAM3 서버 시작4090 GPU에서 실행
$cd ~/lekiwi_sam3/4090 && python sam3_server.py
▶ 이렇게 하면

4090 GPU에서 SAM3 정밀 분할 서버가 포트 5001에서 대기해요.

🔍
SAM3의 역할

YOLOE가 "여기에 병이 있다"는 사각형 박스를 주면, SAM3는 그 안에서 병의 정확한 윤곽을 픽셀 단위로 그려줘요. 사각형에는 배경이 20% 정도 포함되는데, SAM3 마스크는 물체만 정확히 분리합니다.

🧠 Step 3: VLM 서버 시작 (4070)

VLM은 이미지를 보고 "이 물체가 정상인가? 결함이 있나?"를 판단하는 AI 두뇌입니다

5🧠
VLM 서버 시작Qwen2.5-VL-7B 8bit 양자화
$cd ~/vlm && python vlm_server.py
▶ 이렇게 하면

4070 GPU에서 VLM 서버가 포트 5002에서 대기해요. 모델 로딩에 2~3분 걸려요.

1Qwen2.5-VL-7B 모델이 8bit 양자화로 로드됩니다 (~9GB VRAM)
2"✅ VLM 로드 완료" 메시지가 나오면 준비 완료!
📌8bit vs 4bit: 8bit(~9GB)는 4bit(~5GB)보다 디테일 인식력이 높습니다. 미세한 찌그러짐, 작은 스크래치를 감지하려면 8bit를 권장합니다.

🎛️ Step 4: NUC 오케스트레이터 시작

NUC가 지휘자 역할 — YOLOE, SAM3, VLM을 순서대로 호출합니다

6🎛️
오케스트레이터 시작YOLOE + 웹 서버 + 파이프라인
$cd ~/vlm && python app_scenario3.py
▶ 이렇게 하면

NUC에서 웹 서버가 포트 5000에 열리고, 브라우저에서 품질검사를 할 수 있어요.

1YOLOE-26 OpenVINO 모델이 로드됩니다
2Pi 카메라에서 ZeroMQ로 프레임을 수신합니다
3http://192.168.0.65:5000 에서 웹 UI가 열립니다

🔍 Step 5: 품질검사 실행

브라우저에서 물체를 검사하고 결과를 확인합니다

7🌐
브라우저 접속같은 네트워크의 PC/태블릿에서
$http://192.168.0.65:5000
▶ 이렇게 하면

AI 물류 품질검사관 웹 UI가 열려요.

1왼쪽에 YOLOE 실시간 감시 화면이 보입니다
2카메라 앞에 물체(병, 컵, 상자 등)를 놓으세요
3[🔍 검사 시작] 또는 [🔬 정밀] 버튼을 클릭하세요
8🔍
일반 검사 (🔍)전체 물체를 1회 검사
[🔍 검사 시작] 버튼 클릭 → 약 15초 소요
▶ 이렇게 하면

YOLOE→SAM3→VLM 3단계 파이프라인이 순서대로 실행돼요.

1Step 1 YOLOE (130ms): 물체를 탐지하고 영어 라벨을 붙여요 (bottle, cup...)
2Step 2 SAM3 (1초): YOLOE 라벨로 정밀 마스크를 만들고 배경을 제거해요
3Step 3 VLM (10초): 배경 제거된 물체를 확대해서 손상 여부를 판단해요
9🔬
정밀 검사 (🔬)상/중/하 3등분 검사
[🔬 정밀] 버튼 클릭 → 약 45초 소요
▶ 이렇게 하면

물체를 상단/중앙/하단 3등분하여 각각 VLM이 검사해요.

1SAM3 cutout을 3등분하여 각 영역을 1024px로 확대
2VLM이 각 영역을 개별적으로 검사 (VLM x3회 호출)
3한쪽면만 찌그러진 미세한 결함도 감지 가능!
🔬
왜 3등분인가?

VLM에게 "위 중앙 아래 다 봐!"라고 하면 혼란스러워합니다. 대신 "여기만 봐!"라고 한 부분씩 보여주면 집중해서 결함을 찾아요. 사람도 병 전체를 한 번에 보는 것보다 위/중간/아래를 나눠서 보면 더 정확하잖아요.

⚠️프롬프트 팁: "추가 질문" 입력란에 "찌그러짐 확인"처럼 구체적으로 쓰면 VLM이 해당 결함에 집중합니다. "상태 확인해주세요"처럼 모호한 질문은 정확도가 떨어져요.

🔑 핵심 개념 정리

⚡ YOLOE-26 = 눈

Open Vocabulary로 텍스트 설명만으로 어떤 물체든 탐지. 80종 고정이 아니라 원하는 물체를 자유롭게 지정.

🎯 SAM3 = 확대경

YOLOE 박스에서 물체만 정밀 분리 (배경 제거). 이 cutout을 확대하면 표면 디테일 극대화.

🧠 VLM = 검사관 두뇌

"정상은 이렇다"는 기준과 비교하여 스크래치, 찌그러짐, 변형을 한국어로 설명.

📐 확대의 균형

1024px = 결함+정상 구분 최적. 2048px = 너무 디테일해서 제조 특성도 결함으로 오인 가능.

💬 프롬프트 엔지니어링

"정상 기준" + "무시할 것" + "찾을 것"을 명확히 지시해야 AI가 정확하게 판단합니다.

🏭 공장 배포 시

동일 제품이므로 SAM3_LABEL_MAP + 고해상도 카메라 + 72B 모델로 정밀도를 높일 수 있습니다.
