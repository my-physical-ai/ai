총 파라미터848M · 가중치 약 3.5GB (GPU 추론 전제)
① 비전 인코더Perception Encoder 기반 ViT — 약 450M
② 텍스트 인코더Perception Encoder 텍스트부 (BPE) — 약 300M

④ 추적기SAM 2 Tracker (메모리 뱅크)
추론 속도~30ms/이미지 (100객체, H200) · HF facebook/sam3

FathomNet — 수중 생물 분할

FathomNet(MBARI)은 SAM 3 이미지 분할 모델로 수중 생물 마스크를 생성합니다. 아래 SA-Co 데이터셋 예시처럼, SAM 3는 다양한 텍스트 개념(고양이, 자동차, 과일, 전자기기 등)을 픽셀 단위로 분할합니다. 같은 방식이 탁한 물속 심해 생물에도 적용됩니다.

데이터 라벨링 · MLOps
Roboflow Auto-Label — 텍스트로 라벨링

가장 강력한 실무 활용. "solar panel" 텍스트 하나로 항공 사진의 패널을 전부 분할합니다. 이렇게 만든 라벨로 경량 YOLO/RF-DETR을 학습시켜 GPU 없는 현장에 배포합니다. (실제 결과 이미지는 Roboflow 블로그 링크 참고 — 보라=정상 패널, 빨강=눈 덮인 패널처럼 표시됩니다.)

👨‍🏫
선생님 → 학생 (지식 증류)

SAM 3(무겁지만 똑똑한 선생님)가 라벨을 만들고, YOLO(가볍고 빠른 학생)가 현장에서 실시간 추론. NUC에서도 돌아갑니다.

🤖 사용 모델SAM 3 (라벨러)YOLOv8/11-segRF-DETR
# SAM 3로 세그멘테이션 데이터 자동 라벨링 — PyTorch / transformers from transformers import Sam3Model, Sam3Processor # SAM 3 모델·프로세서 import torch # 추론용 model = Sam3Model.from_pretrained("facebook/sam3").to("cuda").eval() # 모델 로드(3.5GB) processor = Sam3Processor.from_pretrained("facebook/sam3") # 전처리기 inputs = processor(images=image, text="solar panel", return_tensors="pt").to("cuda") # 이미지+텍스트 with torch.no_grad(): # 추론 모드 outputs = model(**inputs) # SAM 3 추론 results = processor.post_process_instance_segmentation(outputs, threshold=0.5)[0] # 마스크 후처리

🖼️ solar/vehicle/box 결과 blog.roboflow.com ❄️ 눈 덮인 패널 탐지 blog.roboflow.com ▶ 직접 이미지 올려보기 playground.roboflow.com

로보틱스 · 피지컬 AI
자연어 물체 인식 → 매니퓰레이션

로봇은 SAM 3의 오픈 보캐뷸러리로 "파란 컵을 집어"처럼 말로 물체를 지정합니다. 특히 Meta는 Aria Gen 2 스마트글래스의 1인칭 영상에서 "hands"로 손을 분할·추적하는 실제 데모를 공개했는데, 이는 로봇 시점(egocentric) 인식에 직결됩니다. Llama MLLM과 결합한 SAM 3 Agent는 "사람이 손 뻗는 물체 찾아" 같은 복합 추론도 처리합니다.

자율주행
희귀 객체 인식 — "vehicle" 분할

자율주행은 고정 라벨에 없는 물체가 위험합니다. SAM 3는 "vehicle" 하나로 거리의 차·오토바이를 전부 분할합니다(아래 위 1번 데이터셋 이미지에서도 자동차가 분할됨). 실제 차량엔 SAM 3로 라벨을 만들고 경량 모델을 증류해 올립니다.

9
의료 영상
병변 분할 (파인튜닝 필수)

SAM 3는 "platelet(혈소판)" 같은 세밀한 의료 개념은 기본 상태로 약합니다. Meta도 이를 명시하며, 소량 데이터로 파인튜닝하면 빠르게 적응한다고 밝혔습니다. (Roboflow 파인튜닝 가이드의 "cardboard/trash" 예시로 파인튜닝 후 분할 흐름을 볼 수 있습니다.)
